# Chaerum Portfolio

2026 포트폴리오

## 구조

- `index.html` — 메인 (보드 네비게이션)
- `mystory.html` — MY STORY
- `work.html` — Work
- `menu.html` — 메뉴
- `style/` — 공통 레이아웃 스타일
- `assets/css/` — 폰트·페이지 스타일
- `assets/js/` — 메인 스크립트 (GSAP 등)
- `assets/img/` — 이미지 (mystory, work 등)

## 사용 방법

- **간단히**: `index.html`을 브라우저에서 열기
- **로컬 서버로 보기** (필요 시): `npx serve .` 또는 `npm run start`
